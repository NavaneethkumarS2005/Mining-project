# Mining Problem Forecast System and Risk Analyzer

A web-based project designed to present mining-related information and basic risk awareness using a simple and interactive user interface. This project focuses on frontend development concepts using HTML, CSS, and JavaScript.

## Overview

This project contains multiple web pages that display mining-related content, images, and layouts. It is intended as an educational and demonstration project to understand how frontend technologies can be used to build informational systems related to mining and safety awareness.

## Features

- Multi-page website
- Clean UI design
- Image-based visual sections
- Basic JavaScript interactivity
- Organized layout using CSS

## Tech Stack

- HTML
- CSS
- JavaScript
